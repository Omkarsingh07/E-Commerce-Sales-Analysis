# Superstore Sales Data Analysis Project

## Project Overview

This project focuses on analyzing Superstore sales data using Python and Plotly to uncover business insights related to sales, profit, discounts, customer segments, shipping trends, and product performance.

The analysis helps understand:
- Sales performance
- Profitability trends
- Customer behavior
- Regional performance
- Product category analysis
- Impact of discounts on profit

# Data Preprocessing

The following preprocessing steps were performed:

- Converted `Order Date` and `Ship Date` into datetime format
- Extracted:
  - Order Month
  - Order Year
  - Day of Week
- Checked dataset information and summary statistics
- Handled encoding issues using latin-1 encoding

# Key Insights

- Technology category contributes significantly to overall sales.
- High discounts often lead to negative profits.
- Certain regions outperform others in profitability.
- Consumer segment contributes the highest sales.
- Some furniture products generate losses despite high sales.
